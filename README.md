# 🚀 LifeForge – Habit Tracker App

LifeForge is a habit tracking app designed to help users stay consistent and improve their daily routines. It focuses on building discipline through simple tracking, streaks, and progress visualization.

The goal is to make self-improvement easier to follow and more engaging in daily life.

## 📱 Features

### 🌳 Habit Tracking

• Create and manage daily habits
• Mark habits as completed
• Track progress automatically

### 🔥 Streak System

• Build daily streaks
• Streak resets if a habit is missed
• Encourages consistency

### ⏱ Focus Mode

• Timer options:
• 30 minutes
• 1 hour
• 2 hours
• Helps users stay focused during work or study
• Shows session summary after completion

### 📊 Progress Tracking

• View consistency over time
• Track completion rate
• Simple activity insights

### 🎯 Daily Rewards

• Earn points for completing habits
• Maintain streaks to get rewards
• Unlock achievements

### 🔒 Premium Features

• Unlimited habits
• Advanced statistics
• Customization options
• Additional features for power users

### 💾 Data Persistence

• Data is saved locally or in the cloud
• No reset on refresh or reopening the app

## 📦 Installation

To install the app on your Android device:

1. Convert the web app into an APK using an online APK builder
2. Download the APK file
3. Transfer it to your phone
4. Enable "Install from Unknown Sources" in settings
5. Install and open the app
